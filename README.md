## Sobre o projeto

Pokédex desenvolvida em grupo como projeto acadêmico do curso de ADS.

A aplicação consome a **PokéAPI** e exibe informações dos Pokémon filtrados por tipo, com navegação entre páginas, barra de busca e seção de notícias sobre o universo Pokémon.

O projeto foi construído com **HTML, CSS e JavaScript puro**, sem uso de frameworks.

---

## Funcionalidades

- 🏠 Página inicial com notícias e DLCs recentes da franquia
- 🔍 Busca de Pokémon por nome
- 🪨 Listagem por tipo: Rock, Ghost, Dragon e Steel
- 👥 Página da equipe de desenvolvimento

---

## Tecnologias

- HTML5
- CSS3
- JavaScript
- [PokéAPI](https://pokeapi.co/) — API pública consumida para trazer os dados dos Pokémon

## Como executar localmente

Por ser um projeto em HTML puro, basta clonar e abrir no navegador:

```bash
git clone https://github.com/DaviNunes01001/Pok-dex_Pro.git
```

Abra o arquivo `index.html` diretamente no navegador, ou use a extensão **Live Server** no VS Code.
